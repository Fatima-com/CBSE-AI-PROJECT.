# NODAL, THE ELITE CBSE MENTOR.
This is a personal project made by Fatima, aimed to aid class 12 CBSE students in their course. It is an AI-integrated CBSE competency dashboard built with Python and Streamlit. It has RAG-inspired mentoring logic to assist CBSE class 12 students with the massive NCERT syllabus.
# DESCRIPTION OF THE PROJECT
The project contains a high performance AI dashboard named NODAL designed to help class 12 CBSE students master competency-based questions in various streams such as Science, Commerce or Humanities.
# FEATURES
It has dedicated environments for streams such as Commerce, Science and Humanities.
It is powered by Llama 3.3 for NCERT-focused guidance
It is built to bridge the gap between textbook theory and competency based questions of the cbse class 12 board exam.
# BUILDING BLOCKS
Frontend: Streamlit (Python)
Brain: Groq Cloud API (Llama 3.3)
